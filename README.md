## Running the Project

### Environment Variables
Please make sure you have the following files set up with the necessary variables before running the project:

- `.env` in the **backend** directory  
- `.env.local` in the **frontend** directory  

---

### Backend Setup
1. Open a VS Code terminal
2. Navigate to the backend directory:
   ```bash
   cd backend
3. Run the backend script:
   ```bash
   python georeference.py

### Frontend Setup
1. Open a second VS Code terminal
2. Navigate to the frontend directory:
   ```bash
   cd frontend
3. Install dependencies:
   ```bash
   npm install
4. Start the development server:
   ```bash
   npm run dev

Once both the backend and frontend are running, the application should be accessible locally.
